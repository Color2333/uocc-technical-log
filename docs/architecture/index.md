# 系统架构总览

## 整体组成

```
                         水下信道（浑浊水体）
                               ▲
                               │ 光信号
                               │
         ┌─────────────────────┴─────────────────────┐
         │                  发射端                    │
         │  ┌─────────┐    ┌─────────┐    ┌───────┐  │
         │  │ 文字编码 │───►│ OOK     │───►│ RLL   │  │
         │  │ UTF-8   │    │ 调制    │    │ 编码  │  │
         │  └─────────┘    └─────────┘    └───────┘  │
         │                          │                │
         │                          ▼                │
         │                   ┌─────────────┐          │
         │                   │ LED 驱动    │          │
         │                   │ 10W + 3.3W  │          │
         │                   └─────────────┘          │
         └────────────────────────────────────────────┘
                               │
                               │ 光
                               ▼
         ┌────────────────────────────────────────────┐
         │                  接收端                      │
         │  ┌─────────┐    ┌─────────┐    ┌─────────┐  │
         │  │ 双目    │───►│ 增益    │───►│ OOK    │  │
         │  │ 相机    │    │ 自适应  │    │ 解调   │  │
         │  └─────────┘    └─────────┘    └─────────┘  │
         │                                    │       │
         │                                    ▼       │
         │  ┌─────────┐    ┌─────────┐    ┌─────────┐  │
         │  │ MIMO   │───►│ RLL    │───►│ 文字    │  │
         │  │ 合并   │    │ 解码   │    │ 输出   │  │
         │  └─────────┘    └─────────┘    └─────────┘  │
         └────────────────────────────────────────────┘
```

## 信号流

```
文字输入 → UTF-8编码 → OOK调制 → RLL编码 → LED驱动 → 光信号
                                                          ↓
光信号 → 相机采集 → OOK解调 → RLL解码 → MIMO合并 → 文字输出
```
