# 清水信道实验

## 实验目的

验证在水体透明、无浑浊影响的情况下系统的基线性能。

## 实验条件

| 参数 | 值 |
|------|---|
| 环境 | 透明水箱 |
| 距离 | 3m |
| 水体 | 透明（无添加物）|
| 速率 | 10 kbps |

## 实验状态

🔄 进行中

## 预期目标

清水信道 BER 应接近空气信道，证明水体本身对光学信号无显著衰减。

## 待验证指标

| 指标 | 目标 |
|------|------|
| BER | < 10⁻⁴ @ 3m |
| 端到端延迟 | < 50ms |
| 自适应收敛时间 | < 15 帧 |

## 实验步骤

1. ✅ 水箱准备
2. ✅ 设备架设
3. 🔄 基线测量
4. 🔄 数据记录
