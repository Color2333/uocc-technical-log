# 浑浊水信道实验

> *核心创新点验证*

## 实验目的

验证自适应控制算法在动态浑浊环境下的有效性。

## 实验条件

| 参数 | 值 |
|------|---|
| 环境 | 水箱（添加牛奶/泥沙）|
| 距离 | 3m |
| 浑浊度 | 可调（低/中/高）|
| 速率 | 10 kbps |

## 实验状态

🔄 待开始

## 这是核心验证实验

浑浊水信道实验是验证本系统"感知-反馈"自适应闭环的核心实验：
- 固定参数系统在浑浊水下 BER 急剧上升
- 自适应系统应能通过 ISO 增益/曝光时间调节维持稳定通信

## 预期结果

| 指标 | 固定参数系统 | 自适应系统 |
|------|------------|-----------|
| 低度浑浊 BER | < 10⁻³ | < 10⁻⁴ |
| 中度浑浊 BER | < 10⁻² | < 10⁻³ |
| 高度浑浊 BER | 通信中断 | < 10⁻² |

## 验证步骤

1. 🔄 配置浑浊剂（牛奶/泥沙）
2. 🔄 标定浑浊度等级
3. 🔄 固定参数系统 baseline
4. 🔄 自适应系统对比测试
5. 🔄 记录收敛时间、稳态 BER

## 关键指标

- **自适应收敛时间**：从信道变化到重新稳定的时间
- **稳态 BER**：长期运行后的误码率
- **通信中断次数**：在测试周期内通信中断的次数
