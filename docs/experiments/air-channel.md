# 空气信道基准实验

## 实验目的

建立室内空气环境的基准性能指标，用于对比后续水信道实验。

## 实验条件

| 参数 | 值 |
|------|---|
| 环境 | 室内空气 |
| 距离 | 1m / 3m / 5m |
| 速率 | 10 kbps |
| 调制 | OOK |
| 功率 | 10W LED |

## 实验结果

### 误码率 (BER)

| 条件 | 结果 | 状态 |
|------|------|------|
| 1m, 10kbps | < 10⁻⁶ | ✅ |
| 3m, 10kbps | < 10⁻⁴ | ✅ |
| 5m, 10kbps | 待测 | 🔄 |

### 端到端延迟

| 指标 | 值 |
|------|---|
| 10kbps 端到端延迟 | < 50ms |

### 距离范围

| 条件 | 结果 |
|------|------|
| 直线可视 | 5m 内稳定通信 |

## 分析

空气信道下系统性能优异，BER 满足通信要求。5m 距离仍有进一步测试空间。
